# 逆序输出字符串

输入一个字符串，然后逆序输出它。

### 输入格式

一行字符串。

### 输出格式

一行逆序的字符串。

### 数据范围

0 ＜ len(S) ≤ 1000

#### 输入样例 1

```
a
```

#### 输出样例 1

```
a
```

#### 输入样例 2

```
abcdef
```

#### 输出样例 2

```
fedcba
```

#### 输入样例 3

```
1234cba
```

#### 输出样例 3

```
abc4321
```

### 题解

```c
void puts_rev(char *s) {
  // 找到字符串的长度
  int len = 0;
  while (s[len] != '\0') {
    len++;
  }
  // 从字符串的末尾开始，逆序输出每个字符
  for (int i = len - 1; i >= 0; i--) {
    putchar(s[i]);
  }
}
```
